# Drox IDE 1.5.1

## Nouveautés

- **Fil de discussion chronologique** aligné sur le TUI : phases, outils inline, réponse sous `answering`.
- **Wizard « Connect your AI »** : connexion cloud ou serveur perso, headers HTTP personnalisés.
- **Replay session** compatible avec le nouveau fil ; correction du busy après Stop.
- **Cartes `ask_user`** : rendu markdown, hauteur bornée.
- Libellés de phase en **anglais** dans le chat (Reasoning, Reading, Work, …).

## Statut

Toujours **expérimental**. Release **Windows** uniquement ; Linux prévu en **1.5.3**.

## Prérequis

- Windows 10 ou plus récent (64 bits)
- [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/) (souvent déjà installé)
- [Ollama](https://ollama.com/) ou API compatible (configuration via le wizard)
