# Drox IDE 1.5.3

## Nouveautés

- **Diffs fichier** dans le fil : modifications inline, undo et redo par carte.
- **Cartes shell** : commandes bash / PowerShell avec sortie structurée (stdout, stderr, code de sortie).
- **Messages utilisateur** : copie, expansion du texte long, rappel commit discret.
- **Composer** : zone de saisie auto-agrandissante (2 à 12 lignes), pleine largeur du panneau.
- **Style fil TUI** : police VT323, verts phosphore, alignement au bord du panneau.
- **WORK strip** : plan stable, compteurs reasoning, grille d’activité nettoyée après les runs.
- **Historique sessions** : liste épurée, rejeu UI complet à la réouverture.
- **Popup nouveautés** au premier lancement (cliquer sur la version dans l’en-tête chat pour rouvrir).
- **Base VS Code** : intégration **1.126.0** sur la ligne `main` sources.

## Prérequis

- Windows 10 ou plus récent (64 bits)
- [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/) (souvent déjà installé)
- [Ollama](https://ollama.com/) ou autre serveur LLM compatible (wizard « Connect your AI »)
