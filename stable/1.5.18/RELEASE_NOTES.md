# Drox IDE 1.5.18

## Nouveautés

- **Reprise de conversation** — à la réouverture de l’app, la dernière session est restaurée (plus de chat vide systématique)
- **Chat IDE fiable** — fin du blocage « Loading session… » ; l’historique se charge depuis le dossier workspace
- **Stop intelligent** — avant réponse : le message revient dans l’input ; en cours : la réponse partielle reste affichée
- **Éditer le dernier message** — dialog **Keep partial** / **Discard & restart** après une interruption

## Prérequis

- Windows 10 ou plus récent (64 bits)
- [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/) (souvent déjà installé)
- Serveur LLM : [Ollama](https://ollama.com/) ou API OpenAI-compatible
