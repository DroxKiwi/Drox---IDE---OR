# Drox IDE 1.3.3

## Nouveautés

- Release fiable Windows : installeur aligné sur le code source (bundle UI 1.3.2+ et moteur MODERN).
- Garde-fous de build : vérification du package avant publication.
- Chat Drox modulaire (lazy history, contributions D1.x).
- Menu Aide Drox, télémétrie Microsoft désactivée par défaut.

## Prérequis

- Windows 10 ou plus récent (64 bits)
- [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/) (souvent déjà installé)
- [Ollama](https://ollama.com/) pour le chat local (recommandé)
