# Drox IDE 1.5.5

## Correctifs (juin 2026)

- **Fil de discussion** : ordre chronologique strict dans `#log` — plus de superposition WORK / plan / réponses après plusieurs tours.
- **Scroll** : un seul défilement dans le fil ; reprise de session et chargement d’historique sans fil figé ni « double fenêtre ».
- **Reprise session** : envoi de message débloqué après rechargement d’une conversation.
- **Diffs fichier** : repli / dépli fonctionnel ; bouton de disclosure harmonisé sur outils, shell et phases.
- **Chargement** : grille 3×3 et phrases geek dès l’envoi du message (plus de vide avant la réponse).
- **UI** : marge bas de fil pour les languettes agent ; padding des cartes shell aligné sur les diffs.

## Nouveautés

- **Historique session** : pagination « charger plus » sans casser l’ordre du fil.
- **Base VS Code** : **1.127.0** (inchangée depuis 1.5.4).

## Prérequis

**Windows** : Windows 10+ 64 bits · [WebView2](https://developer.microsoft.com/microsoft-edge/webview2/) · [Ollama](https://ollama.com/) ou LLM compatible

**Linux** : Ubuntu 22.04 / 24.04 amd64 · dépendances graphiques standard · Ollama ou LLM compatible
