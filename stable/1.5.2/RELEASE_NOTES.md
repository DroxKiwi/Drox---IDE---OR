# Drox IDE 1.5.2

## Nouveautés

- Configuration moteur alignée sur le contrat `tui_mono` / `agent.run`.
- Panneau **Architecte** : sampling complet + `keep_alive` en release.
- Panneau **Général** : `max_iterations` (défaut 50).
- Purge des réglages legacy 1.4 (`engine.tuning.*`, orchestration `role_split`).
- Popup « What's new » au premier lancement ; clic sur la version dans le header chat pour rouvrir les notes.

## Prérequis

- Windows 10 ou plus récent (64 bits)
- [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/)
- Serveur LLM (Ollama, vLLM, etc.) configuré via le wizard connexion
